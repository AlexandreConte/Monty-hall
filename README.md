# Projeto Next.Js utilizando TypeScript

## Tela inicial
Escolhe-se uma quantidade de portas a serem renderizadas no aplicativo, e qual porta deverá conter o prêmio, após a escolha, clicar em Start para gerar o jogo.

## Tela do jogo
Mostra as portas, o número da porta, ao selecionar uma porta, ela altera seu estilo e com detalhes em amarelo, ao selecionar outra porta, a porta anterior é desmarcada e a próxima recebe a seleção.
Ao clicar na maçaneta da porta, abre a porta, revelando se ela continha o prêmio.
Clicar em Reset game volta para a tela inicial
